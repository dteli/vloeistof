About Vloeistof

Vloeistof is a way to keep track of cocktails you enjoy and want to create.

You can add your own custom cocktails or search from a database of already created cocktails.

This project was built by Eli T. Drumm, Reese Dick, and Lindsey Dickerson as a requirement for
Eleven Fifty Academy's Red Badge portion of the October 2019 Javascript Cohort.

Want to know more about our project and requirements? Look at our README on Github!
